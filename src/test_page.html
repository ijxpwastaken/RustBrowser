Rust Browser Engine

A browser with JavaScript, React, and Database support!

JavaScript Engine

Full JavaScript support including:

  • Variables, functions, loops
  • Objects and arrays
  • console.log, Math, alert
  • DOM APIs

React Support

Basic React APIs available:

  • React.createElement
  • React.useState
  • React.useEffect
  • ReactDOM.render
  • ReactDOM.createRoot

Database Support

Database APIs available:

  • MySQL connections
  • SQLite support
  • IndexedDB
  • fetch API
  • Promise support

Rust Browser Engine v1.0

Supports: HTML, CSS, JavaScript, React, Database, Network
